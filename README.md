### 抢票效果
运行代码后，Splinter 会自动启动浏览器，代替手工去操作页面，执行打印日志如下。
```
请输入验证码...
开始购票...
第1次点击查询...
第2次点击查询...
第3次点击查询...
第4次点击查询...
第5次点击查询...
第6次点击查询...
第7次点击查询...
第8次点击查询...

预订失败...
开始预订...
开始选择用户...
提交订单...
确认选座...
预订成功...
抢票通知邮件发送成功！
```
抢票成功后，我们会收到抢票成功的邮件。
同时，火车票订单中有一张刚才待支付的火车票，手动支付成功后就完成整个抢票流程了。
